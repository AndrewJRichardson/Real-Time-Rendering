Detailed Description

Interface class for creating a rasterizer mode.

Rasterizer modes define how the transformed vertices of an object get drawn to screen

Rasterizer Modes are attached to objects so that each object can be drawn using a different mode if the user wishes

This is not pixel shader implementation, though you could implement a pixel shader within a rasterizer mode

Override operator() to create a rasterizer mode

Responsible for ZBuffering - hope to fix this in a later update
